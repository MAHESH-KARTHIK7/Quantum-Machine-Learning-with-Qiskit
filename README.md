 Quantum Machine Learning with Qiskit

Overview
    This project implements a hybrid Quantum-Classical Machine Learning model using Qiskit and QSVC for binary classification.

Features
- Quantum Feature Mapping
- QSVC Classifier
- Data Preprocessing
- Model Evaluation
- Decision Boundary Visualization

Technologies Used
- Python
- Qiskit
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

How to Run
1. Install required libraries.
2. Run the Python script.
3. View classification results and plots.

Results
- Accuracy: 92%
- Precision: 93%
- Recall: 92%
- F1 Score: 92%
